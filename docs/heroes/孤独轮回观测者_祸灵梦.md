# 孤独轮回观测者 · 祸灵梦

**主属性**：智力

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | +1.0 |
| 智力 | 40 | +1.0 |

## 技能

???+ abstract "Q　· 祸符　冷却：10秒　3级解锁"

    向前方丢出符箓，对范围内敌人造成一次**智力×15**混乱伤害。

???+ abstract "W　· 吞噬魂灵　冷却：30秒　10级解锁"

    消耗周围 **800码**内友军英雄（每人消耗其 **50% 当前生命**，若该英雄智力<500则生效）；被消耗的友军会受到 **智力-500** 惩罚，持续3秒。每消耗 1 名友军，为自身增加全属性 +60；固定额外 +100 全属性，持续约 **7秒**；施放时同时对 **500码**内敌人造成 **智力×32.5** 混乱/毁灭伤害。

    !!! tip "W→E联动"
        释放W后，E技能会从被动切换为**主动施法**形态（冷却10秒）：向前方发射阴阳玉，对路径 **250码**内敌人造成 **智力×1.3** 多段伤害（每0.15秒），持续3秒。施放主动E后自动切回被动形态。

???+ abstract "E　· 黑雷阴阳玉　冷却：被动　15级解锁"

    攻击有**20%**概率抛出阴阳玉，对目标周围约**250码**敌人造成**智力×5**混乱伤害并**眩晕1秒**。

    !!! tip "触发器验证"
        概率实际为 `GetRandomInt(0,100)<=20`（约19.8%），范围250码。

???+ abstract "R　· 黄昏·万劫崩坏　冷却：30秒　20级解锁"

    展开黄昏崩坏大阵，对阵中敌人每**0.1秒**造成约**智力×8.5**伤害；技能持续期间会不断消耗自身最大蓝量，蓝空自动结束。

???+ abstract "第5技　· 轮回　冷却：50秒　35级解锁"

    释放崩坏能量，对**1200码**内敌人造成**130×智力**混乱伤害，同时为1200码内友军英雄提供**全属性+200、护盾+30000、攻击+10000**，持续6秒。

    !!! tip "触发器验证"
        施放后英雄不会死亡，不消耗复活次数。伤害含1.3倍隐藏系数（实际为智力×100×1.3）。
