# 驯兽师 · 西莉卡

**主属性**：智力

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | +1.0 |
| 智力 | 20 | +1.25 |

!!! note
    西莉卡的技能以**智力**为伤害基础（描述中写作"体力"）。

## 技能

???+ abstract "Q　· 龙音　冷却：8秒　3级解锁"

    指向**敌人**：毕娜盘旋敌人头顶，造成5段**智力×10**混乱伤害；指向**友军（含自身）**：毕娜盘旋头顶，恢复5段**智力×10**的生命值。

???+ abstract "W　· 冰冻霜晶　冷却：16秒　10级解锁"

    向目标位置射出多枚冰晶，命中敌人造成**智力×35**混乱伤害并冰冻1秒。

???+ abstract "E　· 疾风幻蝶　冷却：16秒　15级解锁"

    快速多段攻击使敌人无法行动，最终伤害计算为**智力×130**混乱伤害。

???+ abstract "R　· 华娜冰晶　冷却：34秒　20级解锁"

    毕娜向扇形范围持续喷射冰晶，持续约3秒，每0.5秒造成**智力×25**混乱伤害，同时为周围800码友军恢复该次伤害**×0.1**的生命值。

???+ abstract "D　· 快速支援　冷却：12秒　初始自带"

    使自身短暂脱离敌方视野并迅速切入；随后下一击造成**智力×22**混乱伤害。
