# 冬弥（泳装形态）

**主属性**：智力

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.5 |
| 敏捷 | — | — |
| 智力 | 20 | +1.5 |

!!! note
    泳装冬弥的所有技能均以**智力**为伤害基础（技能描述中写作"体力"）。

## 技能

???+ abstract "Q　· 幽魂戏水　冷却：20秒　3级解锁"

    召唤幽魂随行，持续**15秒**；每次普攻时幽魂同步攻击目标，造成**智力×5**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"体力×3"，实测为**智力×5**。

???+ abstract "W　· 水上烟花　冷却：14秒　10级解锁"

    向目标区域射出6枚烟花，1秒后对**300码**范围敌人造成**智力×30**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"×50体力"，实测无加成道具时**智力×30**（见下方装备加成）。

???+ abstract "E　· 水浪冲击　冷却：20秒　15级解锁"

    释放6道水弹呈扇形涌出，将前方敌人持续击退；每弹命中造成**智力×25**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"体力×25"，实测为**智力×25**；描述写"CD间隔:25"，实测为**20秒**。

???+ abstract "R　· 水枪射击　冷却：24秒　20级解锁"

    向前方连续发射水枪，距离越远伤害越高；近端**智力×80**至远端（约900码）**智力×110**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"最高伤害×60体力"，实测最高为**智力×110**，且属性为**智力**而非体力。
