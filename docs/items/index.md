# 物品图鉴

> **数据来源**：item.ini（对象数据）+ ability.ini（技能数据）+ war3map.j（合成/掉落逻辑）
> 所有数值均以实际代码为准，部分物品描述文本存在未及时更新的情况，Wiki 以代码实测值为准。

## 分类导览

| 分类 | 说明 |
|------|------|
| [素材](素材/index.md) | 无战斗属性，用于打造装备或出售 |
| [武器](武器/index.md) | 主手武器，直接影响攻击力与基础属性 |
| [副武器](副武器/index.md) | 副手装备，提供防御、攻速或特殊被动 |
| [装备道具](装备道具/index.md) | 首饰/配件类，装备于道具槽（最多2件） |
| [装甲](装甲/index.md) | 身体防具，提供防御力、生命值及特殊效果 |
| [头部道具](头部道具/index.md) | 头部防具，提供多种属性与特殊技能 |
| [不归类](不归类/index.md) | 灵魂武器、专属角色特殊道具等 |
| [消耗物品](消耗物品/index.md) | 药水、食物、永久增益书等一次性物品 |
| [特殊](特殊/index.md) | 传送卷轴、任务道具、NPC交互道具等 |

## 阅读说明

| 字段 | 说明 |
|------|------|
| **物品ID** | 物品的内部编号，如 I000、I00A |
| **品质** | 劣质 / 普通 / 优越 / 稀有 / 传说 / 神具 / ★~★★★★★ |
| **层数** | 名称前缀「X层」或说明中标注的推荐层段 |
| **筋力** | 影响物理伤害与生命值的核心属性 |
| **敏捷** | 影响攻击速度与闪避的核心属性 |
| **体力** | 影响魔法值与部分特殊效果的核心属性 |

!!! note "关于「无法获取」标注"
    标题中含有 **(无法获取)** 的物品，经核查在触发器代码中**无任何关联逻辑**。这类物品可能是历史版本遗留或开发中取消的设计，游戏内**无法通过任何正常渠道获得**，仅记录属性数据供参考。
