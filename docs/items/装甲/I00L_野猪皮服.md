# I00L · 野猪皮服


> **来源层**：1层（Level 11）　**品质**：劣质　**售价**：50金

**基础属性**

| 属性 | 数值 |
|---|---|
| 筋力 | +15 |
| 生命值上限 | +150 |
| 魔法防御 | -13%（减少受到魔法伤害13%）|

| 技能名 | 类型 | 效果 |
|---|---|---|
| 生命恢复 Lv1 | 被动 | 每秒恢复生命值（低量持续回血）|

- 描述：能具有保护性质的防具皮具
- **获取**：1城防具打造，消耗[野猪的皮](../素材/I000_野猪的皮.md)时进入 `ChooseRandomItemExBJ(11, ITEM_TYPE_POWERUP)` 装甲池，随机产出。
