# I0GV · 雷暴战甲

> **品质**：传说

**基础属性**

| 属性 | 数值 |
|---|---|
| 筋力 | +300 |
| 敏捷 | +300 |
| 生命值上限 | +12000 |
| 防御力 | +50 |
| 闪避 | +45% |

**技能效果**

| 技能名 | 效果 |
|---|---|
| 镜面反射 | 受到攻击将对敌人进行敏捷×3的伤害反弹 |
| 雷神之体 | 不受寒冷(30-40层HP减少)和炎热(30-40层HP减少)和地藤(50-60层缠绕)的影响 |
| 负面免疫 | 收到伤害时免疫所有负面状态影响 |

- 描述：传说中的雷兽所掉落的物品，强大无比，具备超强的防御能力，任何一切攻击都能很好的防御。
- **获取**：Boss 掉落。击杀 `Lv85:雷罚龙`（单位 `n023`）时固定掉落。
