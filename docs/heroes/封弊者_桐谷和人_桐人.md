# 封弊者 · 桐谷和人（桐人）

**主属性**：敏捷

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | — |
| 智力 | 20 | +1.0 |

## 技能

???+ abstract "Q　· 愤怒刺击　冷却：7秒　3级解锁"

    高速冲向目标，造成**力量×(15\~23)**的混乱伤害并眩晕1.5秒。

    !!! warning "描述与实测差异"
        技能描述写"筋力×13\~23"，实测为**×15\~23**。

???+ abstract "W　· 四方斩　冷却：12秒　10级解锁"

    向四个方向各射出一道剑影（最大距离650码），命中敌人造成**敏捷×23**混乱伤害；释放后W键切换为「极速移动」，可与四方斩影子换位并对影子周围造成**敏捷×18**范围伤害。

???+ abstract "E　· 星爆气流斩　冷却：35秒　15级解锁"

    化身虚影、自身无敌，对周围300码范围敌人连续**16刀**，每刀造成固定200+**敏捷×10**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"固定250+每刀×22敏捷"，实测为**固定200+×10敏捷**。

???+ abstract "R　· 日蚀　冷却：40秒　20级解锁"

    闪现至目标身旁，连续**27刀**每刀造成**敏捷×7**混乱伤害；最后一刀造成**敏捷×100**伤害。

    !!! warning "描述与实测差异"
        技能描述写"每刀×4敏捷"，实测为**×7敏捷**。

???+ abstract "D　· 招架（剑术格挡）　冷却：30秒　初始自带"

    施放后进入短暂无伤状态；实测免伤约**3.1秒**。同时桐人常驻被动为“造成伤害×2”。

!!! note
    开局即自带W切换机制（无需升级）：初始可用的是「四方斩」（`A1B0`），施放后临时切到「极速移动」（`A1AY`），二段结束或超时后再切回。

## 专属装备

**黑剑·阐释者 / 黑剑·阐释者(真)**（仅桐人可装备，由血盟-桐人Lv100掉落/合成）

装备后，R技能「日蚀」发生以下变化：

- 连击数从 **27刀** 增加至 **35刀**
- 每刀伤害从 **敏捷×7** 提升至 **敏捷×10**
- 最后一刀伤害从 **敏捷×100** 提升至 **敏捷×103**

!!! warning
    技能描述说"携带加成武器变为三十六刀"，实测为**35刀**。
