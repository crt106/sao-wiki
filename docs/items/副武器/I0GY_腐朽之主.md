# I0GY · 腐朽之主

> **品质**：礼物　**类型**：副武器

**基础属性**

| 属性 | 数值 |
|---|---|
| 体力 | +330 |
| 攻击速度 | +100% |
| 移动速度 | +120 |
| 生命值上限 | +8000 |

**技能效果**

| 技能名 | 类型 | 效果 |
|---|---|---|
| 高级致命 Lv5 B级 | 被动 | 物编能力 `A0LK`，装备时由专属触发器授予 |
| 隔绝斩击 C级 | 被动 | 物编能力 `A08K`，装备时由专属触发器授予 |
| 吸血鬼的病原体 Lv5 B | 被动 | 物编能力 `A0F2`，装备时由专属触发器授予 |
| 专属限制 | 装备限制 | 仅指定赞助/定制玩家可装备；非指定玩家拾取后会被移除 |

- 描述：曾经的腐朽之主,不曾被历史所记载,只有活在那个大陆的人才会知道过去的往事
- **获取**：特殊渠道。拾取 `I0KV` 后，若满足 `udg_zhiyuanpin[7] == 1` 且玩家为指定赞助 ID，会发放 `I0GY` 与 `I081`；普通地图游玩中不可稳定获取。
