# 天堂

*头衔：双叶杏* · **主属性**：敏捷

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | +1.0 |
| 智力 | 20 | +1.0 |

!!! note
    天堂的技能以**力量**为主要伤害基础（描述中写作"体力"）。

## 技能

???+ abstract "Q　· 魔法精灵　冷却：2秒　3级解锁"

    释放精灵至目标：目标为**友军**时治疗固定300+**力量×3**的生命值；目标为**敌人**时造成等量伤害（虚无伤害）。

???+ abstract "W　· 破坏光线　冷却：10秒　10级解锁"

    召唤魔法元素攻击敌人，造成固定300+**力量×7**的魔法伤害。

???+ abstract "E　· 魔法激光　冷却：25秒　15级解锁"

    发射激光对目标造成持续魔法伤害（固定100+**智力×3**），同时使目标**无法行动5秒**。

???+ abstract "R　· 你的名字　冷却：10秒　20级解锁"

    在指定地点召唤陨石雨，持续**15秒**；范围内友军每次被陨石治疗固定300+**力量×2**的生命值。⚠️**负面效果**：施法者自身在持续期间不断损失等同于**力量×1**的生命值。

???+ abstract "D　· 魔法契约　冷却：15秒　初始自带"

    与一名友军签订契约；此后自身释放技能时，契约者位置会产生破坏光线。
