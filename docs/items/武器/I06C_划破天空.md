# I06C · 划破天空


> **品质**：神具

**基础属性**

| 属性 | 数值 |
|---|---|
| 攻击力 | +15,000 |
| 生命值上限 | +10,000 |
| 攻击速度 | +100 |

**技能效果**

| 技能名 | 效果 |
|---|---|
| 强行夺取(划破天空) | 攻击造成伤害500+全属性×1的生命恢复,对敌人造成1000+敏捷×20的伤害 |

- 描述：被封印着许久的武器，它来至于人间，在某片大陆曾经黑暗笼罩着天空，大地失去光芒，恶魔妖怪游走人间，是它划破了那黑暗的天空，光芒重新照射大地，它的主人也无人知晓
- **获取**：前奏曲·狼神 Lv58 掉落（必定）
