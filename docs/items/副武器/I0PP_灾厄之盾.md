# I0PP · 灾厄之盾

> **品质**：传说

**基础属性**

| 属性 | 数值 |
|---|---|
| 筋力 | +400 |
| 敏捷 | -200 |
| 智力 | -200 |
| 生命值上限 | +30000 |
| 护甲 | +150 |

**技能效果**

| 技能名 | 效果 |
|---|---|
| 灾厄 | 自身将多承受15伤害，但会将自身承受的10伤害反弹给伤害来源者,秒杀类伤害无效 |
| 能抗最毒的打 | 增加自身5万生命 |

- 描述：灾厄之盾是一件传说中的黑暗系防御装备，以"吞噬灾厄、反噬敌人"为核心特性，能将承受的部分伤害转化为诅咒或腐蚀性能量反弹给攻击者。
- **获取**：未发现正常获取途径。2.3 触发器仅注册 `-cxitem` 描述，未发现 Boss 掉落、宝箱、打造、扭蛋或合成引用。
