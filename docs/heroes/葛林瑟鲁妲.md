## 技能

???+ abstract "Q　· 盾击　冷却：7秒　3级解锁"

    以盾猛击单体，随机触发 **1/4** 档位伤害（CHAOS毁灭）：STR×5 / STR×10 / STR×15 / STR×20，各档等概率。眩晕目标约 **3秒**（Storm Bolt 基础效果）。

???+ abstract "W　· 勇风之息　冷却：被动　10级解锁"

    受击时回复生命，回复量约为**力量×(2~5)**（随机）。

???+ abstract "E　· 圣光十字剑　冷却：18秒　15级解锁"

    在前方约 150 码处创建十字圣光，对 **350码**范围内所有敌人造成**力量×30**混乱/毁灭伤害；持有专属 `I0CC` 时改为**力量×50**。

???+ abstract "R　· 勇往直前　冷却：30秒　20级解锁"

    向目标地点突进，被撞到的敌人承受多段**力量×5**伤害，并波及周围**350码**敌人且击晕**3秒**；与亚丝娜联动时伤害更高。

???+ abstract "D　· 向前突击　冷却：10秒　初始自带"

    朝前方高速突进，期间可闪避伤害与攻击。
