**技能效果**

| 技能名 | 物品描述 | 触发器实测 |
|---|---|---|

!!! warning "描述与实际不符"
    - **绯梦伤害系数虚标**：描述写「筋力×3/段」，触发器实际仅 **筋力×1/段**（两段合计 STR×2），相差 3 倍。
    - **破灭之痕伤害类型**：描述写「魔法伤害」，实际是普攻的附加伤害（100% 触发 +1000），跟随普攻的攻击/伤害类型，并非独立魔法伤害。
