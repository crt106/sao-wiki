# 公会：命运之夜（CD）

**主属性**：智力

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | +1.0 |
| 智力 | 20 | +1.0 |

## 技能

???+ abstract "Q　· 切割（炎斩）　冷却：**5秒**　3级解锁"

    *技能ID：A0ED*

    单体攻击。默认造成**力量×8**混乱/神圣伤害（"炎斩"），显示字样随机；若持有专属物品 `I0LK`（灵魂·鸢）则改为造成**全属性×8**混乱/神圣伤害（"爆炎斩"）。

???+ abstract "W　· 影子　冷却：**35秒**　10级解锁"

    *技能ID：A0EE*

    在自身位置创建一个**镜像分身**；分身无技能、攻击力与防御力均为本体的**50%**，持续 **3分钟**。

???+ abstract "E　· 疯狂幻想　冷却：**42秒**　15级解锁"

    *技能ID：A0EF*

    对目标施放持续性连击。**第一阶段**：连续攻击约 **28次**，每次造成**敏捷×5**混乱/全能伤害，攻击间隔逐渐加快；**第二阶段**：接续 **24次**全属性连击，每次造成**(力量+敏捷+智力)×7**混乱/全能伤害。

???+ abstract "R　· 狮子之弹　冷却：被动　20级解锁"

    *技能ID：A0EI*

    **30%概率暴击**（伤害 **×3.5**）；**20%概率闪避**攻击；每次普攻对目标额外造成**力量×5**毁灭伤害。

???+ abstract "D　· 精神分裂　冷却：**25秒**　初始自带"

    *技能ID：A0EC*

    持续 **7秒**，攻击速度 **+100%**、所受伤害提高，同时无视物理攻击。
