# I0O7 · 染血的小刀

> **品质**：传闻

**基础属性**

| 属性 | 数值 |
|---|---|
| 攻击力 | +3000 |
| 攻击速度 | +200% |
| 生命值上限 | +20000 |

**技能效果**

| 技能名 | 效果 |
|---|---|
| 染血 | 攻击触发后造成自身当前生命值 × 0.05 的混乱/火焰伤害，触发器冷却 3 秒 |
| 锋利 | 攻击百分5概率无视闪避造成3倍伤害 |
| 逐影 | 攻击时概率召唤一个虚影，同步英雄攻击力，只能存在一个；`-cxitem` 写 1%，触发器判定为随机 0-100 小于等于 15 |

- 描述：一把禁止的刀刃，它上面存在一些血迹，仿佛在告诉别人他的来历。佩戴上它能让你感觉到无穷的力量，那种让人越战越疯狂的力量。
- **获取**：Boss 掉落。击杀 `★★★★★Lv70:战皇★★★★★`（单位 `n042`）时固定掉落。
