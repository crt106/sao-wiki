# 未来初音

*头衔：为世界卖萌* · **主属性**：智力

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | +1.0 |
| 智力 | 20 | +1.0 |

## 技能

???+ abstract "Q　· 大葱敲击　冷却：5秒　3级解锁"

    对目标敌人进行敲击，造成**(智力×3)+萌音数值**毁灭伤害，之后眩晕0.5秒。每当使用一次大葱敲击，萌音数值会增加50点。

???+ abstract "W　· 歌之扩散　冷却：10秒　10级解锁"

    歌声传递之时，如果传递给敌人时，造成**(智力×3)+萌音数值**混乱伤害；如果传递给队友时，队友生命值会恢复**(智力×3)+萌音数值**。

???+ abstract "E　· 救赎之声　冷却：25秒　15级解锁"

    每秒对附近500范围内的敌人进行净化，每次净化造成**(智力×15)+萌音数值**混乱伤害，持续5秒。

???+ abstract "R　· 钟声回响　冷却：35秒　20级解锁"

    多次对范围敌人造成伤害，初始范围600码、伤害**力量×10**；随后伤害和范围逐渐提升，伤害最大提升到**力量×50**，范围最大到650码。

## 专属装备

**玲和连的礼物 (I0AR)**（硬专属）

- 获取方式：玩家镜音处购买
- 效果：未来初音达到 **30级** 时拾取，自动发放**连心手环 (I01T)**（STR+50、AGI+50、INT+50）

!!! note
    物品描述写35级，实际触发判定为 **30级**。
