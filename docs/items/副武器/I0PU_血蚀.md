# I0PU · 血蚀

> **品质**：传说

**基础属性**

| 属性 | 数值 |
|---|---|
| 生命值上限 | +22000 |
| 魔法值上限 | +6000 |
| 全属性 | +320 |

**技能效果**

| 技能名 | 效果 |
|---|---|
| 能力转换 | 主动使用时在力量/敏捷/体力形态间轮换，对应授予 `A1L2`、`A1L3`、`A1L4` |
| 强行夺取 | 攻击时按当前形态计算对应主属性 × 10，并回复 `500 + 该数值` 生命，CD 3 秒 |

- 描述：是鲜血与黑暗的结晶，它不只是一件武器，更是与你共生的嗜血魔物，唯有最强大的意志，才能驾驭这蚀骨的力量，在血与火的洗礼中，成就不朽的传说。
- **获取**：合成获得。由 [I0G4 · 骷髅指挥官之剑](I0G4_骷髅指挥官之剑.md) + [I0M9 · 边缘魔石](../装备道具/I0M9_边缘魔石.md) 合成。
