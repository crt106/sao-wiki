# 加速世界 · 黑雪姬

**主属性**：智力

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | +1.0 |
| 智力 | 20 | +1.0 |

!!! note
    黑雪姬的技能以**智力**为伤害基础（描述中写作"体力"）。

## 技能

???+ abstract "Q　· 空间模拟　冷却：8秒　3级解锁"

    利用虚拟空间制造光速攻击，对敌人造成**智力×3**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"体力（智力）×5"，实测为**×3**。

???+ abstract "W　· 模仿数据　被动　10级解锁"

    **被动**：SAO开发者能力，回避**80%的物理攻击**（无冷却，持续生效）。

    !!! warning "描述与实测差异"
        在游戏内显示为「数据模仿」，ability.ini 实际名称为「模仿数据」。

???+ abstract "E　· 模拟空间2　冷却：15秒　15级解锁"

    当所有光线集中时对周围产生多次**智力×8**混乱伤害；若有敌人处于光线正中心则额外**眩晕3秒**。

???+ abstract "R　· 黑色蝴蝶　冷却：18秒　20级解锁"

    制造虚拟爆破，命中敌人造成**智力×25**混乱伤害。
