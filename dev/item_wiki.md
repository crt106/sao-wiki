## 阅读说明

| 字段 | 说明 |
|---|---|
| **物品ID** | 物品的内部编号，可用于查询相关信息 |
| **品质** | 劣质 / 普通 / 优秀 / 稀有 / 史诗 / SSR / 特定（专属）|
| **层数** | 名称前缀「X层」或说明中标注的推荐层段 |
| **筋力** | 影响物理伤害与生命值的核心属性 |
| **敏捷** | 影响攻击速度与闪避的核心属性 |
| **体力** | 影响魔法值与部分特殊效果的核心属性 |

> **关于「无法获取」标注**：标题中含有 **(无法获取)** 的物品，经核查在 `war3map.j` 触发器代码中**无任何关联逻辑**（不出现在掉落表、合成配方、商店初始化等任何触发器中）。这类物品可能是历史版本遗留、开发中取消或被替换的设计，游戏内**无法通过任何正常渠道获得**，仅记录其属性数据供参考。
