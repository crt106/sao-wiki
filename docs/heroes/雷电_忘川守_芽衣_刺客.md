## 技能

???+ abstract "Q　· 三途枯榷　冷却：6秒　3级解锁"

    闪身到敌人背后发动斩击，获得**1点「残梦」**，并为目标附上**1层「集真赤」**；本次斩击造成约**所有攻击力×1.3**的混乱伤害。

???+ abstract "W　· 四相断我　冷却：12秒　10级解锁"

    闪到敌阵中连续发动**3段**斩击，对约**500码**内敌人每段造成**所有攻击力×4**混乱伤害；同时获得**2点「残梦」**并追加**2层「集真赤」**。

???+ abstract "E　· 红叶时雨·万倾一空　冷却：被动　15级解锁"

    当「残梦」达到**9点**时，下一次攻击会触发八连斩；每段约为**所有攻击力×2.2**混乱伤害，收刀再补一次**所有攻击力×6**的毁灭伤害，并清空残梦。

???+ abstract "R　· 残梦尽染·一刀缭断　冷却：28秒　20级解锁"

    先对指定区域约**500码**内造成**6次所有攻击力×3**的混乱伤害；每消去一层「集真赤」会追加伤害并继续抬升下一段倍率，最后再发动一次**所有攻击力×8**的毁灭终结。

**专属武器**：行于流逝的岸 `I0Q3`（攻击力+16000，魔抗+50，防御+80，HP+20000）

| 被动 | 效果 |
|------|------|
| 司渡 | 「红叶时雨·万倾一空」和「残梦尽染·一刀缭断」伤害提升约**18~24%** |
| 司命 | 致命伤害时**锁1血→无敌3秒→恢复满血**，CD 120秒 |
| 忘川 | 主动：对周围**450码**敌人造成**攻击力×20**混乱伤害，CD 100秒 |
