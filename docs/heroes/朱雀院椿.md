# 朱雀院椿

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 30 | +1.5 |

## 技能

???+ abstract "Q　· 敌意感知　冷却：7秒　3级解锁"

    以自身为中心连续3段震荡（每段间隔约0.3秒），对500码内敌人依次造成**力量×6 / ×12 / ×18**混乱伤害。

???+ abstract "W　· 火焰吹熄　冷却：10秒　10级解锁"

    连续4波范围爆破（每波间隔约0.5秒），对500码内敌人造成**力量×18 / ×15 / ×18 / ×15**混乱伤害。

???+ abstract "E　· 疯狂幻想　冷却：42秒　15级解锁"

    锁定单体后进入连续斩击阶段：前段高频连击每次造成**敏捷×5**混乱伤害；连击累积到阈值后进入强化阶段，按节奏追加多段**全属性和×7**混乱伤害。

???+ abstract "R　· 惩戒·圣火　冷却：36秒　20级解锁"

    持续约7秒召唤圣火打击，每0.1秒触发一轮；每轮在随机落点对550码敌人造成**力量×(1~3)**混乱伤害（多段可叠加）。
