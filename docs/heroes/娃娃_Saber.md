## 技能

???+ abstract "被动　· —　被动"

    每次普通攻击额外附带**力量×2**混乱伤害（始终有效）。

???+ abstract "Q　· 风王结界　冷却：7秒　3级解锁"

    瞬移至目标区域，在1秒内高速闪现**10次**，每次闪现对**350码**范围内敌人造成**敏捷×2**混乱伤害；若范围内敌人≤10个追加**敏捷×1**伤害；结束后返回原位。

    !!! warning "描述与实测差异"
        技能描述写"筋力×1/×3"伤害，实测代码为**敏捷×2**（基础）+ **敏捷×1**（≤10敌人时追加）。

???+ abstract "W　· 对魔力　冷却：13秒　10级解锁"

    自身在**3.5秒**内所受魔法伤害完全无效化。

???+ abstract "E　· 誓约胜利之剑　冷却：35秒　15级解锁"

    向前方约960码放出宽**300码**光束，持续**3.2秒**；每0.2秒对光束内敌人造成**(力量+敏捷)×10**混乱伤害。

???+ abstract "R　· 遥远的理想乡　冷却：32秒　20级解锁"

    在自身位置创造理想乡（持续10秒）；自身每秒回复**力量×30**生命（共5次）；立即回复**600码**内友军各**力量×30**生命；激活期间每次普通攻击额外追加**智力×5 + 敏捷×8**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"攻击追加筋力×5 + 敏捷×8"，实测为**智力×5 + 敏捷×8**（`GetHeroInt`）。
