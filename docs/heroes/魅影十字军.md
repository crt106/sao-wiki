# 魅影十字军

**主属性**：力量

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 40 | +1.0 |
| 敏捷 | 40 | — |
| 智力 | 40 | +1.0 |

## 技能

???+ abstract "Q　· 战炎之怒　冷却：8秒　3级解锁"

    向目标方向释放冲击波，命中终点范围内敌人造成**攻击力×1.5 + 力量×18**混乱伤害，并眩晕目标，将其牵拉至中心位置；3秒内目标移速降低60%，自身对该目标造成的伤害提升10%。

    **强化版（R一段命中后的4秒窗口内再次施放）**：技能改为调用E技能的位移斩击模式，伤害升级为 E 技能强化版公式。

    !!! warning "描述与实测差异"
        技能描述写"最大生命值×1.1 + 力量×15"，实测触发器为**攻击力×1.5 + 力量×18**。

???+ abstract "W　· 不朽之辉　冷却：15秒　10级解锁"

    自身所受伤害减少 **40%**，持续 **5秒**。

???+ abstract "E　· 雷霆震怒　冷却：9秒　15级解锁"

    以贝塞尔曲线弧线跳跃至目标位置，落地对范围450内敌人造成**攻击力×4.0 + 力量×70**混乱伤害并眩晕3秒。

    **强化状态（R二段触发的4秒E强化窗口内）**：伤害提升 **×3.5倍**，即约**攻击力×14.0 + 力量×245**。

    !!! warning "描述与实测差异"
        技能描述写"攻击伤害×1.5 + 力量×50，强化提升250%"，实测触发器基础伤害为**攻击力×4.0 + 力量×70**，强化倍率×3.5（与描述的250%提升系数一致，但基础值不符）。

???+ abstract "R　· 一段剑舞：执鞘 → 二段剑舞：掷剑　冷却：12秒 / 1秒　20级解锁"

    **一段 · 执鞘**（消耗100蓝，CD：12秒）：对路径敌人造成**攻击力×2.0 + 力量×35**混乱伤害；同时刷新战炎之怒（Q）的冷却，开启4秒Q强化窗口，并解锁R二段。

    **二段 · 掷剑**（消耗0蓝，CD：1秒，执鞘命中后激活）：对周围范围内敌人造成**攻击力×3.25 + 力量×80**混乱伤害；触发后开启4秒E强化窗口，并使自身每秒恢复**10000**生命值，持续6秒。

    !!! note "连携说明"
        R一段 → R二段之间有4秒窗口；R二段命中后再有4秒E强化窗口。完整连携顺序为：R一段 → R二段 → E强化（或强化Q）。

???+ abstract "F　· 救赎·怒剑狂花　冷却：50秒　35级解锁"

    终极技能。释放两段主打，每段对前方扇形范围造成**攻击力×6.0 + 力量×60**混乱伤害，最终一击追加**力量×150**混乱伤害并眩晕3秒。

    !!! warning "描述与实测差异"
        技能描述写"两刀攻击伤害×5 + 力量×45，最后一击+力量×125"，实测触发器为**每段×6.0攻击力+力量×60，最终一击力量×150**。

???+ abstract "D　· 剑术奥义　冷却：15秒　初始自带"

    **被动**：每次普攻有 **18%** 概率触发狂暴，额外造成**攻击力×1 + 力量×10**混乱伤害。

    **主动**（消耗50蓝）：接下来 **3次攻击** 附加狂暴强化效果，持续10秒。

!!! note "Combo 机制总结"
    魅影的核心套路围绕 R → R二 → E 的顺序展开，Q可在R一的4秒窗口内升级为跳斩版本。各窗口均为 **4秒**，节奏较快。

## 其他说明

魅影十字军为 2.41 版本新加入英雄，自带新型位移系统（贝塞尔曲线跳跃）与多段 Combo 窗口机制。
