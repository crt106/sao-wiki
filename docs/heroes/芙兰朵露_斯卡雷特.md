# 芙兰朵露·斯卡雷特

*头衔：尾随姐姐的怪妹子*

## 基础属性

| 属性 | 初始值 | 每级成长 |
|------|--------|---------|
| 力量 | 20 | +1.0 |
| 敏捷 | 20 | +1.0 |
| 智力 | 20 | +1.0 |

## 技能

???+ abstract "Q　· 禁弹「Starbow Break」　冷却：6秒　3级解锁"

    向目标方向射出**3颗弹丸**（间隔30°散射），每颗飞行**650码**；命中敌人时造成**敏捷×12**混乱伤害，并将其拉回施法者身边。

???+ abstract "W　· 禁忌「Four of a Kind」　冷却：23秒　10级解锁"

    锁定目标，双方暂停行动**5秒**；期间每0.15秒出现幻影分身轮番冲击：一套造成**力量×7**混乱伤害，另一套造成**敏捷×7**穿刺伤害；5秒结束后追加终结一击，造成**目标自身力量×15**混乱伤害。

    !!! warning "描述与实测差异"
        技能描述写"每次敏捷×7混乱伤害，终结一击敏捷×15"，实测每0.15秒有两套分身同时攻击——力量×7混乱 + 敏捷×7穿刺；终结一击伤害为**目标方自身的力量×15**（非施法者属性）。

???+ abstract "E　· 禁忌「Kagome Kagome restrict」　冷却：30秒　15级解锁"

    将目标绑缚约1.5秒，自身在其周围高速连砍**10次**，每次造成**敏捷×16**混乱伤害。

???+ abstract "R　· 血影杀　冷却：25秒　20级解锁"

    在周围350\~550码随机位置生成**7道血影**，各自冲向目标造成**(力量+智力)×4**混乱伤害；血影攻击后自身冲刺至目标再造成**敏捷×23**混乱伤害，共8击。

    !!! warning "描述与实测差异"
        技能描述写"8次均为敏捷×23混乱"，实测前7道血影每影伤害为**(力量+智力)×4**，仅最后自身冲刺一击为**敏捷×23**。

???+ abstract "第5技　· 禁弹「Starbow Break」终极　冷却：33秒　35级解锁（扩展位）"

    自身**无敌**；每0.15秒在500码范围释放一道冲击波造成**力量×5**混乱伤害，共7道；最终引爆**600码**范围造成**力量×60**混乱伤害。
