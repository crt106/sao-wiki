# I0P3 · 元素之茅

> **品质**：特殊

**基础属性**

| 属性 | 数值 |
|---|---|
| 体力 | +150 |
| 生命值上限 | +10000 |

**技能效果**

| 技能名 | 类型 | 效果 |
|---|---|---|
| 体力恢复 | 被动（Lv5） | 物编能力 `A1EF`；2.3 触发器未注册额外专属逻辑 |
| 元素之力 | 主动（A级，CD:12） | 物编能力 `A0XO`；2.3 触发器未注册额外专属逻辑 |

- 描述：在57层所在的环境下凝练而成的武器，其内部蕴含的多种元素的力量，对于魔法师来说是一件不错的武器。
- **获取**：金色宝箱掉落。指定 `nrvs` 宝箱掉落表固定掉落。
