# 谏山黄泉

!!! note "同名区分"
    本页为游戏内名称「**谏山黄泉**」的**敏捷英雄**。另有两位含「黄泉」名称的英雄：[黄泉（Six）](黄泉.md)、[雷电·忘川守·芽衣（黄泉）](雷电_忘川守_芽衣_刺客.md)。

## 技能

???+ abstract "Q　· 怨魂瞬斩　冷却：**8秒**　3级解锁"

    *技能ID：A1LI*

    蓄力 0.6 秒后，瞬移至指定位置；到达后对 **350码**内敌人造成**敏捷×20 + 1000 + 当前蓝量**混乱/全能伤害，并附加 **5秒眩晕**（怨魂标记）。

???+ abstract "W　· 死亡切割　冷却：**15秒**　10级解锁"

    *技能ID：A1LJ*

    向前方 **1000码**高速突进，沿途对 **350码**范围内敌人造成**敏捷×60**毁灭伤害；**必定**触发 **×1.5～3.5** 随机倍率（条件恒为真）。

???+ abstract "E　· 乱红莲　冷却：**22秒**　15级解锁"

    *技能ID：A1LL*

    在侧方 150码处生成炮台，向前方发射约 **1500码**纵深的弹幕走廊（宽约 **230码**）；每 **0.1秒**对走廊内敌人造成**敏捷×18**混乱/全能伤害，持续 **2.5秒**（共 25 跳）。

???+ abstract "R　· 谏山刀法　冷却：**24秒**　20级解锁"

    *技能ID：A1LK*

    瞬移至目标单位处，双方定身；对目标发动 **16连击**，每击造成**敏捷×25**毁灭伤害。攻击间隔从 0.25 秒起每击缩短 0.01 秒（最快 0.18 秒/击），越到后期越快。

???+ abstract "第5技　· 杀生石　冷却：**25秒**　35级解锁"

    *技能ID：A1LM*

    主动效果：**全属性+300**，攻速与移速提高；持续 **8秒**，期间造成伤害可回复其 **8%** 生命值。

???+ abstract "D　· 怨魂　冷却：**22秒**　初始自带"

    *技能ID：A1LH*

    **被动**：普通攻击有 **10%** 概率为目标附加「怨魂」标记，持续 **4秒**，被标记单位受到的伤害提高 **20%**。（工具提示描述，未在触发器中直接体现）

    **主动**：主动施放时立即**刷新 W「死亡切割」的冷却**。触发器实现仅包含此刷新逻辑（`FlushedAbility(un, 'A1LJ')`）。
